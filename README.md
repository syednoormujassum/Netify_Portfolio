## 🚀 Quick Start

### Local Development

1. **Clone the repository**
```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
cd portfolio
```

2. **Open in browser**
```bash
# Option 1: Direct file access
open index.html

# Option 2: Local server (Python)
python3 -m http.server 8000

# Option 3: Local server (Node.js)
npx http-server
```

3. **Visit in browser**
```
http://localhost:8000
```

### Features to Explore
- 🔗 Click navigation links for smooth scrolling
- 🎨 Hover over project cards for animations
- 📱 Resize browser to test responsive design
- 📞 Check contact section with social links
- 📚 Click "View Project" for detailed information

## 🌐 Deployment

### **Option 1: Netlify (Recommended)** 
```bash
# 1. Push to GitHub
git add .
git commit -m "Portfolio website"
git push origin main

# 2. Go to netlify.com
# 3. Connect your GitHub repo
# 4. Deploy automatically!
```

**Live URL:** `https://your-site.netlify.app`

### **Option 2: GitHub Pages**
```bash
# Create repo: YOUR_USERNAME.github.io
git remote add origin https://github.com/YOUR_USERNAME/YOUR_USERNAME.github.io.git
git push -u origin main
```

**Live URL:** `https://YOUR_USERNAME.github.io`

### **Option 3: Vercel**
```bash
# Push to GitHub, then connect at vercel.com
# Auto-deploys on every push
```

**Live URL:** `https://your-portfolio.vercel.app`

For detailed deployment instructions, see [DEPLOYMENT_GUIDE.md](DEPLOYMENT_GUIDE.md)

## 🔧 Customization

### Update Content
Edit these sections in `index.html`:
- **Hero Section:** Name, designation, description
- **About Section:** Professional summary
- **Skills:** Add/remove skill categories
- **Projects:** Update project details
- **Contact:** Email, phone, location

### Update Colors
Modify CSS variables in `styles.css`:
```css
:root {
    --primary-color: #00d4ff;
    --secondary-color: #0099ff;
    --accent-color: #6a11cb;
    /* ... more variables */
}
```

### Add New Sections
1. Add HTML in `index.html`
2. Style in `styles.css`
3. Add animations in `script.js`
