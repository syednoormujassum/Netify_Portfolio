## 🌐 Deployment

Choose from multiple deployment options:

### **Option 1: Netlify (RECOMMENDED)** ⭐
Easiest and most popular choice with automatic deployments.

```bash
# 1. Create/Push to GitHub
git add .
git commit -m "Portfolio website"
git push origin main

# 2. Go to netlify.com and sign in with GitHub
# 3. Click "New site from Git"
# 4. Select your repository
# 5. Configure build settings (leave blank for static site)
# 6. Deploy!
```

**Advantages:**
- ✅ Free tier with generous limits
- ✅ Auto-deploys on every push
- ✅ Free custom domain
- ✅ Built-in analytics
- ✅ Instant HTTPS

**Live URL Example:** `https://your-site-name.netlify.app`

---

### **Option 2: GitHub Pages** 
Free hosting directly from your GitHub repository.

```bash
# 1. Create repo: USERNAME.github.io
git remote add origin https://github.com/USERNAME/USERNAME.github.io.git
git branch -M main
git push -u origin main

# 2. Go to repository Settings > Pages
# 3. Select 'main' branch as source
# 4. Wait 1-2 minutes for deployment
```

**Advantages:**
- ✅ Free hosting
- ✅ Custom domain support
- ✅ GitHub integration
- ✅ Good for personal projects

**Live URL:** `https://USERNAME.github.io`

---

### **Option 3: Vercel** 
Perfect for static sites and Next.js projects.

```bash
# 1. Push to GitHub
# 2. Go to vercel.com
# 3. Connect GitHub account
# 4. Import your repository
# 5. Deploy!
```

**Advantages:**
- ✅ Free tier
- ✅ Next.js optimized
- ✅ Fast CDN
- ✅ Environment variables support

**Live URL Example:** `https://your-portfolio.vercel.app`

---

### **Option 4: GitHub + Custom Domain**
Use your own custom domain with GitHub Pages.

```bash
# 1. Buy domain from GoDaddy, Namecheap, etc.
# 2. Configure DNS settings
# 3. Add CNAME file to repo:
echo "yourdomain.com" > CNAME
git push

# 4. Go to repository Settings > Pages
# 5. Custom domain field will recognize your domain
```

### Update Colors & Themes

Edit CSS variables in `styles.css` (Lines 1-50):

```css
:root {
    /* Primary Colors */
    --primary-color: #00d4ff;      /* Cyan - Main accent */
    --secondary-color: #0099ff;    /* Blue - Secondary accent */
    --accent-color: #6a11cb;       /* Purple - Tertiary accent */
    
    /* Background Colors */
    --dark-bg: #0a0e1f;           /* Dark background */
    --card-bg: #1a1f3a;           /* Card background */
    --text-light: #e0e0e0;         /* Light text */
    
    /* Other Variables */
    --spacing: 1rem;
    --border-radius: 8px;
    --transition: all 0.3s ease;
}
```

### Responsive Breakpoints

Current breakpoints in `styles.css`:
- **Mobile:** 480px and below
- **Tablet:** 768px to 1024px
- **Desktop:** 1024px and above

Customize media queries:
```css
@media (max-width: 768px) {
    /* Tablet styles */
}

@media (max-width: 480px) {
    /* Mobile styles */
}
```

### Add New Sections

1. **Add HTML** in `index.html`:
```html
<section id="new-section" class="new-section">
    <div class="container">
        <h2 class="section-title">New Section</h2>
        <!-- Your content -->
    </div>
</section>
```

2. **Add CSS** in `styles.css`:
```css
.new-section {
    background: var(--dark-bg);
    padding: 60px 20px;
}

.new-section-content {
    display: flex;
    gap: 20px;
}
```

3. **Add JavaScript** in `script.js`:
```javascript
// Add interactivity for new section
document.querySelector('.new-section').addEventListener('click', () => {
    // Your functionality
});
```
